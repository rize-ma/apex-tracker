# apex-tracker
こちらはAPEXというゲームのプレイヤー情報を検索することができるWEBアプリです
### アクセス
[こちら](https://apex-tracker-nine.vercel.app/)からアクセス可能です  
また、バックエンドのデプロイ先をRenderにしている関係上初回のみリクエストが遅いです  
### 使用技術
##### フロントエンド
- React(Next.js)
- typescript
- mantineUI
- tailwindcss
- tabler/icons-react
- axios
- zustand
- vercel
##### フロントエンド
- node.js(Nest.js)
- axios
